# Calculator-App
it is a calculator app based on html, css and js.

Calculatoor
A modern, scientifically-capable calculator web application featuring a sleek Glassmorphism design.

🌟 Features
1. Glassmorphism UI: Premium aesthetic with semi-transparent backgrounds, blur effects, and glowing interactions aimed at providing a high-quality user experience.
2. Scientific Mode: An expandable side menu (toggled via the hamburger icon) unlocks advanced functions like sin, cos, tan, and power (^).
3. Interactive Animations: Buttons feature responsive hover and active states with smooth scaling and dynamic lighting effects.
4. Smart Error Handling: Visual feedback (red text) for invalid operations or undefined results.
5. "Never Settle" Easter Egg: Try calculating 1+ to trigger a unique animation sequence.
   
🛠️ Technologies Used
HTML5: Structure and layout.
CSS3: Advanced styling using Flexbox, CSS transitions, and backdrop filters for the glass effect.
JavaScript (ES6+): Handles all calculation logic, DOM manipulation, and UI state management.
